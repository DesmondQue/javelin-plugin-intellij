# Javelin Command Setup Instructions

## Option 1: Use the wrapper script from anywhere in javelin-core

From the `javelin-core` directory, you can now run:
```powershell
.\javelin.bat <arguments>
```

Example:
```powershell
.\javelin.bat -t build/classes/java/main -T build/classes/java/test -o report.csv
```

## Option 2: Add javelin-core to PATH (Recommended)

1. Add `C:\THESIS\javelin-core` to your system PATH
2. Then you can run from anywhere:
```powershell
javelin <arguments>
```

### To add to PATH (PowerShell):
```powershell
$env:Path += ";C:\THESIS\javelin-core"
```

### To add to PATH permanently:
1. Open System Properties → Environment Variables
2. Edit the "Path" variable for your user account
3. Add: `C:\THESIS\javelin-core`
4. Restart your terminal

## Option 3: Create a PowerShell alias

Add this to your PowerShell profile (`$PROFILE`):

```powershell
function javelin {
    & "C:\THESIS\javelin-core\javelin.bat" $args
}
```

Then reload your profile:
```powershell
. $PROFILE
```

Now you can use `javelin` from anywhere!

## What was created:

1. **javelin.bat** - Wrapper script in javelin-core directory that:
   - Sets JAVA_HOME to Java 21 automatically
   - Calls the Gradle-generated script
   - Works even when system default is Java 11

2. **build/install/javelin/** - Full distribution with:
   - bin/javelin.bat - Generated by Gradle
   - lib/ - All required JAR dependencies
